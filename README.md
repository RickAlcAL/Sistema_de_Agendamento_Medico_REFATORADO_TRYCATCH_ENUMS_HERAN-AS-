# Sistema_de_Agendamento_Medico_REFATORADO_TRYCATCH_ENUMS_HERAN-AS-
Sistema de agendamento médico refatorado aplicando as melhores práticas de Programação Orientada a Objetos (POO). O projeto foca em robustez e manutenibilidade através da implementação de herança para especialidades, enums para controle de status e tratamento de exceções customizado com blocos try-catch.
